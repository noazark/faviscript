FaviScript by Noah H. Smith

var context = this; var w = context.width; var h = context.height; var centerX = w / 2; var centerY = h / 2; var pad = Math.round(w * 0.10); var radius = (w - pad * 2) / 2; function arc(scale, color) { context.beginPath(); context.arc(centerX, centerY, radius, 0, scale * Math.PI, false); context.fillStyle = color; context.fill(); context.closePath(); } arc(4/3, '#59f');

About

More or less, this is the tool I've always wanted. I hate the process of creating icons for my small weekend projects, and yet, they look so naked without them. Even with a number of image editors at my disposal the whole process annoys me. So, FaviScript was created so I can make my own icons procedurally and never be without an favicon again.
